# ⚡ Quick Start - Get Notes Running in 5 Minutes

Welcome! This guide will get you up and running with the Notes extension in no time.

---

## 1️⃣ Download & Install (2 Steps!)

### Step 1: Download the File
👉 **Click the download button on the GitHub page** or download `Notes.xpi` from the files list

### Step 2: Install in Firefox
1. Open Firefox
2. Drag the `Notes.xpi` file into your Firefox window
3. Click **"Add"** when Firefox asks
4. ✅ Done! You'll see a Notes icon in your toolbar

**That's it!** No complicated setup. No accounts. Just notes.

---

## 2️⃣ Create Your First Note

1. **Click the Notes icon** in your Firefox toolbar
2. A window pops up → Click **+ New Note**
3. **Type a title** (e.g., "My First Note")
4. **Type your content** in the big text area
5. **Click Save**
6. ✅ Your note appears in the list!

---

## 3️⃣ Manage Your Notes

### Edit a Note
- Click on any note in the list
- Make changes
- Click **Save**

### Delete a Note
- Click the trash icon 🗑️ next to a note
- Click **OK** to confirm
- Note is gone

### Go Back to Your List
- Click the **Back** button anytime

---

## ✅ That's It!

Your notes are now safe on your computer. They'll be here even if you close Firefox.

---

## 🆘 Something Not Working?

### Notes extension doesn't show in toolbar
- Restart Firefox completely
- Check if you're looking in the right place (pin it: click the Notes icon → pin icon)

### Can't save notes
- Try refreshing the extension (if testing version: go to `about:debugging` and click reload)
- Check if Firefox storage is enabled (it should be by default)

### Forgot where I saved it
- Go to `about:addons` to see installed extensions
- The Notes extension should be there
